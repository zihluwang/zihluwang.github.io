打开 unRaid 系统中 Docker 的 Remote API

发表于2023-03-22|更新于2023-06-24
|阅读量:

今天需要将一个 Java 项目部署到 Docker 上，奈何本地 Docker 是运行在 unRaid 系统上的。

通过查阅资料，发现开启方法可以使用在 daemon.json 中设置，于是便前往修改 \\$NAS_IP\flash\config\go 文件如下：

mkdir -p /etc/docker

tee /etc/docker/daemon.json <<-'EOF'

{
  "registry-mirrors": [
    "https://docker.mirrors.ustc.edu.cn/"
  ],
  "hosts": [
    "tcp://0.0.0.0:2375",
    "unix:///var/run/docker.sock"
  ]
}

EOF

这样即可打开 Docker 的 2375 Remote API 端口。

2375 Remote API 端口是不需要用户验证的，千万不要在公网环境下打开这个端口！
